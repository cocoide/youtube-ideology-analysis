# シミュレーションデータから得られる示唆

## 分析結果サマリー

シミュレーションデータ（Loss群2動画、Gain群2動画、計427コメント）の分析から以下の結果が得られました：

### 1. 仮説検定の結果

#### H1: 損失フレームは投票宣言（VP）を増やす
- **結果**: 支持される (p = 0.043 < 0.05)
- **効果量**: 0.086 (小さいが有意)
- **Loss群のVP率**: 30.1%
- **Gain群のVP率**: 21.6%
- **解釈**: 損失フレームでは利得フレームより約8.5ポイント多く投票宣言が見られる

#### H2: 利得フレームは外部効力感（E_ext）を増やす  
- **結果**: 強く支持される (p < 0.001)
- **効果量**: -0.195 (中程度の効果)
- **Loss群のE_ext率**: 16.3%
- **Gain群のE_ext率**: 35.8%
- **解釈**: 利得フレームでは損失フレームより約19.5ポイント多く外部効力感が表明される

### 2. 追加の発見

#### シニシズム（Cyn）の差
- **Loss群**: 41.1%（高い冷笑的態度）
- **Gain群**: 18.8%（低い冷笑的態度）
- 損失フレームは同時に高いシニシズムも誘発している

#### 内部効力感（E_int）の類似性
- **Loss群**: 27.3%
- **Gain群**: 23.4%
- フレームによる差は小さい（統計的に有意でない可能性）

### 3. 実践的示唆

1. **メッセージ設計への示唆**
   - 投票率向上が目的なら損失フレーム（「投票しないと損をする」）
   - 政治参加の効力感醸成が目的なら利得フレーム（「あなたの声が届く」）

2. **副作用への配慮**
   - 損失フレームは投票宣言を増やすが、同時にシニシズムも高める
   - この「諸刃の剣」効果は長期的な政治参加意欲に悪影響の可能性

3. **ターゲット層への最適化**
   - 既に冷笑的な層には利得フレームで効力感を高める
   - 無関心層には損失フレームで行動を促す

### 4. 研究の限界と今後の課題

- シミュレーションデータのため、実際の効果量は異なる可能性
- 動画の内容・スタイル・投稿者の影響を統制できていない
- コメントは視聴者の一部のみ（サイレントマジョリティは不明）

### 5. ABMへの示唆

観察された効果を再現するABMパラメータ：
- 損失フレーム感受性（λ）: VP確率を1.4倍に増幅
- 利得フレーム感受性（γ）: E_ext確率を2.2倍に増幅
- シニシズム伝播率: Loss環境で2倍以上

これらの結果は、フレーミング効果が選挙啓発において重要な役割を果たすことを示唆しています。
